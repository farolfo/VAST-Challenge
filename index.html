MC2.3  Which factories are responsible for which chemical releases? Carefully describe how you determined this using all the data you have available. For the factories you identified, describe any observed patterns of operation revealed in the data.

Limit your response to no more than 8 images and 1000 words.

For this question we will build 4 bar charts, one per each chemical. Each chart will have the facotries in the x axis and the chemical's prescence in the air in the y axis.

We will make usage of the wind direction on this calculations (we will not use the wind speed because we only want to see if the wind came from the factory to the sensor's position, so the speed would only tell if the chemical came faster or slower, but it would eventually be present on the sensor).

We will also add a tolerance of 30 degrees, this means that if the wind direction is in the same direction than the vector that goes from the factory to the sensor (with a 15 degrees tolerance per each side), then the chemical reading will be attributed to that factory.
